# Code of Conduct

## Our Standard

We expect contributors to communicate with respect, clarity, and good faith.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or derogatory comments
- Publishing private information without consent
- Repeatedly derailing technical discussions

## Enforcement

Project maintainers may edit, remove, or reject content that violates this code.
Severe or repeated violations can lead to temporary or permanent bans.

## Reporting

Please report violations through GitHub Issues using a private/security-sensitive
channel when appropriate.
